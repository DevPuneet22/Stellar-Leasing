# Product Requirements

## Vision

Stellar Leasing is a workflow automation platform for internal business operations. It should let an admin define a workflow once and run it repeatedly across approvals, checklists, data collection, and SLA-driven follow-up work.

## Target Users

- operations admin
- leasing manager
- approver
- case owner

## MVP Scope

- tenant and user setup
- workflow definition CRUD
- versioned workflow definitions
- manual approval and task steps
- basic conditional transitions
- task inbox
- comments and attachments
- audit trail
- reminder and escalation hooks

## Out of Scope for MVP

- marketplace
- external ERP integrations
- advanced reporting
- billing
- AI-generated workflows

## Core User Stories

- As an admin, I can create a workflow with steps and transitions.
- As a manager, I can approve or reject a step.
- As an operator, I can see my assigned work and due dates.
- As an auditor, I can review who changed what and when.

## First Workflows to Support

- lease approval
- vendor onboarding
- expense approval
